Morerise is an online platform that allows users to quickly learn a new language


Consume

The first step is to learn as much as you can about the situation, the problem at hand, and how you can best serve the client.

Clearly Defining the Problem

According to Language Mastery, the problem with the traditional approach to learning a new language is that "what is easiest and most profitable to design is rarely what is most useful for learners". They claim that these methods lack the authentic input or true output required to help learners make the required associations that will excellerate the learning process. There are many challenges to developing a complete solution, but today we going to focus on the following:

Challenge #1

Limited authentic input or real-time real-life exposure to the language.

Challenge #2

Limited true output or real-time real-life application of the language.


Enhance

After the problem is clearly defined we can look at how others have attempted to solve the problem and where they fell short. Through a shared experience you can start development on a more comprehensive solution.

Competitve Analysis

There are plenty of language learning solutions, Rosetta Stone, Duolingo, and Pimsleur. Rosetta Stone is known for providing real time feedback on pronunciation through audio analysis. Duolingo is known for it's reverse learning approach that teaches english speakers another language and then allows them to study english from the perspective of the other language. Pimsleur is the last solution we will look at and it is known for asking users questions and requesting responces in the language they are learning.

Being realistic, I am not, nor do I want to be, the next Luis von Ahn (duolingo cofounder). That being said, I do see a problem here and believe that a surprisingly simple alteration could alleviate the challenges above. For the sake of brevity, we are ignoring the challenge that direct translation is not an accurate local translation.

Feature #1

Google Translate design allows users to incorperate the application into everyday conversations with a language learning partner.

Feature #2

Speech to text can let users quickly start practicing a new language as inspired by elements from their daily life.

Feature #3

Text to speech can provide insight into correct pronunciation.

Feature #4

A conversation between two people could be held with them both learning a new language or translating one language to another.

Feature #5

As the user starts to practice speaking the new language, speech to text can be used agian to provide real time feedback on pronunciation.


Replicate

Finally, we can attempt to solve the problem through itteration, development, and testing.

Itteration

The prototype for this site was initialy developed on codesandbox.io, like many of my other projects, but ultimately had to be rebuilt as the export refused to respond correctly to ng serve/build.

If one of the aformentioned language learning apps were to implement this idea, I'd recommend keeping the feature locked until a user gets through enough courses to reasonably be able to use the direct translations as a hand rail that accentuates their immersion into the culture.
